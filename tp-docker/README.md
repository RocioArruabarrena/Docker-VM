pwd: muestra la ruta actual
ls: lista archivos y carpetas
cd: cambia de directorio
mkdir: crea carpetas
touch: crea archivos
cat: muestra contenido
cp: copia archivos
mv: mueve o renombra
rm: elimina archivos
chmod: cambia permisos


1. ¿Qué hace el script?
Crea una estructura de carpetas de una empresa, genera archivos de productos, escribe contenido, copia, renombra, elimina archivos y muestra la estructura final.

2. ¿Qué comandos usó?
Todos los vistos en Linux (ls, cd, mkdir, etc.)

3. ¿Para qué sirve chmod +x?
Permite que el archivo se pueda ejecutar como programa dandole los permisos necesarios.

4. ¿Diferencia entre cp y mv?
cp copia archivos
mv mueve o renombra

5. ¿Qué pasa si no uso cd correctamente?
Los archivos se crean en ubicaciones diferentes o el script falla.

6. ¿Qué aprendió?
Lo importante que pueden llegar a ser los comandos de linux bien utilizados en una maquina virtual.
